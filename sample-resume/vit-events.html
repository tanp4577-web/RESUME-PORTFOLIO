VIT Event Portal

Project Overview

A comprehensive event registration and management system. It actively parses ongoing and upcoming event details directly from the official VIT Vellore website to keep data synced, and allows students to register and purchase entry tickets instantly.

Key Features

  • Razorpay Integration: Seamless, secure checkout flow for purchasing event tickets and confirming registrations.
  • Live Data Sync: Automated data collection scripts that fetch ongoing event details directly from the VIT Vellore website.
  • Access Control: Generation of digital entry passes with secure verification for smooth admissions to ongoing events.
