Interactive Weather App

JavaScriptRESTful APIsAsync/Await

Project Overview

A dynamic web application that pulls current and forecast weather data based on geolocation leveraging public weather APIs. The UI seamlessly adapts animations to match the sunny, rainy, or snowy forecast natively through Javascript DOM manipulations.

Technologies Highlight

  • Fetch API with Async/Await control flow.
  • Responsive UI reflecting exact API telemetry responses.
  • Error handling for network timeouts.
